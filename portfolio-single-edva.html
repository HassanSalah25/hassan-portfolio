Edva Crypto Investment Platform

Frontend HTML5, CSS3, and JavaScript
Backend PHP, Laravel
Database MySQL
Purpose Crypto Investment and Wallet Management Platform

Description

Edva is a cryptocurrency investment platform built using Laravel, MySQL, and native HTML/CSS/JavaScript. It allows users to invest in predefined plans, manage their crypto wallets, and perform financial transactions such as deposits, withdrawals, transfers, and earning interest. The platform is designed to ensure secure, efficient, and user-friendly wallet management for crypto investors.

Key Features

  • Crypto Wallet Management:

    Users can manage multiple wallets for deposits, withdrawals, transfers, and interest tracking.

  • Investment Plans:

    Users can invest in different predefined plans to generate profits over time.

  • Secure Transactions:

    All deposits, withdrawals, and transfers are validated and secured using encryption protocols.

  • Interest Generation:

    Automated interest calculation on invested amounts, credited directly to user wallets.

  • Real-Time Dashboard:

    A user dashboard to track balances, transaction history, and investment performance.

  • User Authentication:

    Secure login, registration, and two-factor authentication (2FA) for enhanced account protection.

  • Admin Panel:

    Admins can monitor transactions, manage user accounts, and track investments.

    HTML5, CSS3, JavaScript for responsive and interactive UI components.
